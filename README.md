# YUME — Next-Gen Immersive Website

A Vite + React + TypeScript website for YUME, upgraded with a full immersive dark/neon visual system, animated UI, scroll reveals, glassmorphism cards, ambient background effects, and polished page-by-page experiences.

## Upgraded pages

- **Accueil** — cinematic hero, animated theme rotation, premium sections, counters and CTA blocks.
- **Concept** — full Experience-as-a-Service explanation, tech infrastructure, customer journey, control room, capacity cards and animated system map.
- **Menu** — digital menu experience with search, category tabs, signature products, QR modal and menu-engine storytelling.
- **Galerie** — interactive LED theme library with category filters, autoplay hero preview, visual intensity bars and theme cards.
- **Corporate** — premium event packages, production flow, expansion map, metrics and corporate CTA.
- **Votes** — live community voting page with real-time results, animated progress bars and winner spotlight.
- **Reservations** — multi-step smart booking flow with time slots, table choices, mood selector, live summary and confirmation screen.
- **404** — branded YUME error screen instead of a default page.

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- lucide-react
- React Router

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```
